# Langage et Intelligence Artificielle

Cette page GitHub complète le chapitre "Comprendre les bases des IA génératives et leurs enjeux pour l'enseignement et la recherche" rédigé pour le livre "L'IA et l'IA générative à l'université : Perspectives pluridisciplinaires et prospectives" publié aux Editions Management et Société en 2026, ISBN: 978-2-38630-348-7, disponible à l'achat [en ligne ici](https://editions-ems.fr/boutique/lia-et-lia-generative-a-luniversite/).

L'objectif de cette espace est fournir des ressources complémentaires, des exemples de code et des références pour approfondir les concepts abordés dans le chapitre.

## Contenu de ce dépôt

Pour présenter le contenu du chapitre, 3 notebooks Jupyter sont disponibles dans ce dépôt, chacun abordant un aspect clé du traitement automatisé du langage (TAL ou NLP en anglais) et de l'intelligence artificielle générative (IAG) :

- `1_introduction_aux_traitements_automatises_du_langage.ipynb` : Ce notebook introduit les concepts de base du NLP en distinguant les approches symboliques et les approches statistiques.

- `2_vectorisation_des_textes_et_inference.ipynb` : Ce notebook explore les méthodes de vectorisation des textes, notamment les techniques de sac de mots (bag of words) avec une illustration d'application de l'apprentissage automatique supervisé pour la détection des spams.

- `3_les_modeles_de_langage_et_transformers.ipynb` : Ce notebook présente les modèles de langage, en commençant par les premiers modèles de plongmement des mots et les architectures de type transformers à la base de l'IAG aujourd'hui, incluant un tour d'horizon de la librairie `transformers` de Hugging Face.

## Utilisation des notebooks Jupyter

Les 3 notebooks Jupyter peuvent alternativement être :

- consultés en mode lecture sur GitHub en cliquant sur le nom du fichier en haut de cette page, ce qui permet de visualiser le contenu sans exécuter le code, ou

- exécutés sur Google Colab en cliquant sur le nom du fichier puis en activant l'icône "Open in Colab" qui se trouve dans chacun des notebooks. Un compte Google est nécessaire pour utiliser Google Colab, mais aucune installation locale n'est requise, ce qui facilite l'accès à ces ressources pour tous les utilisateurs.

Pour les utilisateurs souhaitant exécuter ces notebooks localement, il est nécessaire d'avoir Python installé ainsi que les bibliothèques suivantes : `scikit-learn`, `spacy` et  `transformers`.
